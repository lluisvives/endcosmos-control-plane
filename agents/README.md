# EndCosmos Control Plane — Agents

Los agentes son procesos que observan y controlan partes del ecosistema EndCosmos.

## Tipos de agentes

### server-agent
Controla servicios del servidor:
- nginx
- endcosmos-api
- ai-gateway
- database

### ai-agent
Gestiona el gateway de IA:
- embeddings
- modelos
- inferencia

### web-agent
Controla:
- nginx
- despliegues web
- cache

### game-agent
Controla el runtime del juego:
- mundo
- eventos
- bosses
- Create agents system
